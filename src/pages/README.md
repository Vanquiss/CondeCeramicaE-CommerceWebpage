# /src/pages - Páginas de la Aplicación

Páginas principales de la aplicación e-commerce.

## Páginas Disponibles

- **home/** - Página principal con productos destacados y promociones
- **products/** - Listado de productos con filtros y búsqueda
- **product-detail/** - Página de detalle de un producto específico
- **cart/** - Página del carrito de compras
- **checkout/** - Proceso de pago y finalización de compra
- **user-profile/** - Perfil del usuario, datos personales y configuración
- **admin/** - Panel de administración (gestión de productos, pedidos, usuarios)
- **orders/** - Historial de pedidos del usuario

## Estructura de una Página

Cada página típicamente incluye:
- Componente principal de la página
- Lógica de carga de datos
- Integración con servicios/API
- Manejo de estado local
