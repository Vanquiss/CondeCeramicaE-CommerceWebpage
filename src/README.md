# /src - Código Fuente

Este directorio contiene todo el código fuente de la aplicación e-commerce.

## Subdirectorios

- **components/** - Componentes reutilizables de UI
- **pages/** - Páginas/vistas principales de la aplicación
- **layouts/** - Plantillas de diseño compartidas
- **hooks/** - Custom hooks para lógica reutilizable
- **utils/** - Funciones utilitarias y helpers
- **services/** - Servicios de negocio y lógica de aplicación
- **api/** - Backend API (controladores, rutas, modelos)
- **contexts/** - Context API para gestión de estado global
- **styles/** - Estilos globales de la aplicación

## Uso

Importar componentes, hooks y servicios desde aquí en toda la aplicación.
