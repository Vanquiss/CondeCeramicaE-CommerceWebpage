# /src/components - Componentes de UI

Componentes reutilizables de la interfaz de usuario.

## Organización

- **common/** - Componentes comunes (Button, Input, Modal, Card, etc.)
- **products/** - Componentes de productos (ProductCard, ProductList, ProductFilter, etc.)
- **cart/** - Componentes del carrito (CartItem, CartSummary, CartIcon, etc.)
- **checkout/** - Componentes del proceso de pago (CheckoutForm, PaymentMethod, ShippingForm, etc.)
- **user/** - Componentes de usuario (UserProfile, LoginForm, RegisterForm, etc.)
- **navigation/** - Componentes de navegación (Navbar, Sidebar, Breadcrumbs, etc.)
- **header/** - Componentes del encabezado (Logo, SearchBar, UserMenu, etc.)
- **footer/** - Componentes del pie de página (FooterLinks, SocialMedia, Newsletter, etc.)

## Convenciones

- Cada componente en su propia carpeta con archivo index
- Incluir estilos específicos del componente
- Documentar props y uso del componente
