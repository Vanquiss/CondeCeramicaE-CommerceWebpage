# /assets - Recursos Multimedia

Recursos estáticos de la aplicación organizados por tipo.

## Estructura

- **images/** - Imágenes
  - `products/` - Imágenes de productos (alta resolución)
  - `logos/` - Logos de la marca en diferentes formatos
  - `banners/` - Banners promocionales y de portada
  - `categories/` - Imágenes representativas de categorías

- **icons/** - Iconos
  - Iconos SVG para UI
  - Iconos PNG para compatibilidad
  - Favicon y app icons

- **fonts/** - Fuentes tipográficas
  - Fuentes personalizadas de la marca
  - Archivos woff, woff2, ttf

- **videos/** - Videos
  - Videos promocionales
  - Videos demostrativos de productos
  - Tutoriales

## Convenciones

- Usar nombres descriptivos en minúsculas
- Optimizar imágenes antes de agregar
- Mantener múltiples resoluciones para responsive design
- Documentar derechos de autor y licencias
